Grunt

Grunt is a task runner that can perform tasks like minification, compilation, unit testing, linting. Globally install the Grunt command line interface (CLI) and create a Gruntfile.js:


npm install -g grunt-cli
touch Gruntfile.js

Dependencies

In your package.json add these as dev dependencies:


"devDependencies": {
    "grunt": "~0.4.5",
    "grunt-contrib-clean": "*",
    "grunt-htmlhint": "*",
    "grunt-contrib-uglify": "~0.5.0",
    "matchdep": "*"
}

Then run `npm install`

Create a Grunt task


module.exports = function(grunt){

    grunt.initConfig({
        pkg: grunt.file.readJSON('package.json')
    });

    grunt.registerTask('default', []);

};

Grunt

Above grunt.initConfig() add this:


require("matchdep").filterDev("grunt-*").forEach(grunt.loadNpmTasks);

The matchdep module will install any devDependencies from the package.json that begin with "grunt-"

Watch tasks

Inside grunt.initConfig add these options:

htmlhint: {
    build: {
        options: {
            'tag-pair': true,
            'tagname-lowercase': true,
            'attr-lowercase': true,
            'attr-value-double-quotes': true,
            'spec-char-escape': true
        },
        src: ['index.html']
    }
},
watch: {
    html: {
        files: ['index.html'],
        tasks: ['htmlhint']
    }
}

Watch tasks

Now open another terminal and run:


grunt watch

When you edit your HTML file this watch task will run and validate the HTML.

Uglify your code

Have you ever seen minified Javascript? Uglify is a module that compresses your code and is used a lot for production applications. Add this option to grunt.initConfig:


uglify: {
    build: {
        files: {
            'build/js/base.min.js': ['base.js']
        }
    }
},
watch: {
    js: {
        files: ['base.js'],
        tasks: ['uglify']
    },
    html: {...}
}
